# 🧠 QuizMaster Pro – Interactive Knowledge Assessment Platform

## 📖 Project Overview

QuizMaster Pro is a modern, feature-rich web application built with **Python Flask**, **SQLite**, **HTML5**, **CSS3**, and **JavaScript**. It enables users to test their knowledge through interactive quizzes across multiple categories, compete on global leaderboards, earn achievement badges, and track their learning progress with a personalized dashboard. With a beautiful, fully responsive interface and support for **9 languages**, the platform provides an engaging learning experience across desktop, tablet, and mobile devices.

---

## 🎯 Project Objective

The objective of QuizMaster Pro is to create a secure, responsive, and interactive knowledge assessment platform that makes learning enjoyable through quizzes, gamification, and performance tracking while demonstrating modern full-stack web development using Python Flask.
